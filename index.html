Notre vision

Ce projet a été créé par un passionné de vélo pour présenter à son mémoire de fin d'année

Pour les fans

Pour le moment, ce projet présente simplement les étapes du Tour de France 2026. Il vous donnera des informations clés sur chaque étape

Pour l'analyse

Ce projet est créé simplement en HTML CSS et JavaScript pour le moment. Le code est simple pour le moment car pas de logique complexe ni d'API externe.

Pour l'évolution

Vous voyez actuellement la V1, le projet a pour vocation d'évoluer avant le début du Tour de France. J'aimerais ajouter beaucoup plus d'informations avec des API pour suivre le Tour.
